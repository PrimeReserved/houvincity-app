## Features

- **Property Management**: Add, update, and delete property listings. View detailed information about each property, including location, size, amenities, and current status.
- **Tenant Management**: Manage tenant information, including personal details, contact information, and lease agreements.
- **Lease Management**: Create and manage lease agreements. Track lease terms, rental amounts, and payment schedules.
- **Maintenance Requests**: Submit and track maintenance requests. Assign tasks to maintenance personnel and monitor the status of each request.
- **Financial Tracking**: Monitor rental income, expenses, and generate financial reports. Integrate with payment gateways for online rent collection.

## System Requirements

- **Node.js**: Version 14.x or higher
- **NPM or Yarn**: Latest stable version
- **Database**: PostgreSQL or MySQL
- **Browser**: Latest version of Chrome, Firefox, or Edge

## Installation

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/your-username/houvincity.git
   cd houvincity
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Set Up Environment Variables**:
   Create a `.env` file in the root directory and configure the necessary environment variables. Example:
   ```env
   DATABASE_URL=postgres://user:password@localhost:5432/houvincity
   NEXT_PUBLIC_API_URL=https://api.houvincity.com
   ```

4. **Run Database Migrations**:
   ```bash
   npx prisma migrate dev --name init
   ```

5. **Start the Development Server**:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

## User Guide

### Dashboard

The dashboard provides an overview of your properties, tenants, lease agreements, and financial summaries. 

- **Properties Summary**: Overview of all managed properties.
- **Tenant Summary**: Quick access to tenant information.
- **Financial Summary**: Snapshot of income and expenses.

### Property Management

- **Add Property**: Navigate to the properties section and click on "Add Property". Fill in the details and save.
- **Edit Property**: Select a property from the list, make the necessary changes, and save.
- **Delete Property**: Select a property and click on "Delete".

### Tenant Management

- **Add Tenant**: Go to the tenants section, click "Add Tenant", enter the details, and save.
- **Edit Tenant**: Select a tenant, update the information, and save.
- **Delete Tenant**: Select a tenant and click "Delete".

### Lease Management

- **Create Lease**: Navigate to the leases section, click "Create Lease", fill in the lease details, and save.
- **Edit Lease**: Select a lease, make changes, and save.
- **Delete Lease**: Select a lease and click "Delete".

### Maintenance Requests

- **Submit Request**: Go to the maintenance section, click "Submit Request", provide details, and submit.
- **Track Requests**: View the status of all maintenance requests.
- **Assign Tasks**: Assign maintenance tasks to personnel.

### Financial Tracking

- **Income Tracking**: Record rental income and view income reports.
- **Expense Tracking**: Log expenses and generate expense reports.
- **Generate Reports**: Create financial reports for analysis.
